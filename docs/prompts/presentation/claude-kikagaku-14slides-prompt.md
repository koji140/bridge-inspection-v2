# スライドの型
- 1〜5枚目: 要点3〜4個
- 6枚目: As-Is / To-Be（上下フロー）
- 7枚目: 左右2カラム（制約 / 設計）
- 8枚目: フロー
- 9〜11枚目: スクショ枠＋要点3個
- 12枚目: 成果（最大4つ）
- 13枚目: 左右（苦労 / 学び）
- 14枚目: 2セクション（アプリ / 自分）

# 6枚目（最重要：業務フロー図）
- 箇条書きではなく図で出力
- Mermaid形式で出力
構成:
- 上段：As-Isの業務フロー（横長）
- 下段：To-Beの業務フロー（横長）
- 各フローは4つの箱を左→右に並べる
- 上下で同じ構造にする
- テキスト説明は禁止（図のみ）
必ずこの形式で出力:
flowchart TD
  subgraph AsIs[As-Is]
    direction LR
    A1[写真] --> A2[整理] --> A3[報告] --> A4[共有]
  end
  subgraph ToBe[To-Be]
    direction LR
    B1[登録] --> B2[GPT] --> B3[地図] --> B4[出力]
  end
